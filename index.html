FLOW

完整接入流程

01

安装 Telegram 插件

在 Claude Code 中执行安装命令，完成后重载插件。

02

在 BotFather 创建机器人

执行 /newbot，获得 Bot Token。用户名需以 bot 结尾。

03

配置 Token

执行 /telegram:configure <BOT_TOKEN>。若报错，先用 channels 参数重启 Claude。

04

完成配对

去 Telegram 给机器人发一条消息，复制返回的配对命令，在终端执行并确认策略切换。

05

设置后台运行

推荐放进 tmux，避免终端关闭后机器人断线。

RECOVERY

机器人没反应？

  • 确认 Claude 是用 --channels plugin:telegram@claude-plugins-official 启动的
  • 重新执行 /telegram:configure
  • 再去 Telegram 给机器人发消息
  • 拿到配对码后回终端执行

SECURITY

安全与使用建议

  • Bot Token 不要外传
  • 建议将 DM 策略切到 allowlist
  • 电脑需保持开机，网络稳定
  • 后台常驻优先用 tmux
