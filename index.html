🧰 工作流与技巧

计划模式 / 思考强度

  • ShiftTab：Normal → Auto → Plan
  • --permission-mode plan 计划模式启动
  • ultrathink 当轮最大努力度
  • AltT 快 / 平衡 / 深入
  • CtrlO 查看思考
  • /effort low | med | high

Git Worktree

  • --worktree name 每个功能独立分支
  • isolation: worktree 隔离代理
  • sparsePaths 只检出必要目录
  • /batch 自动创建 worktrees

语音模式

  • /voice 开启语音
  • Space 长按录音，松开发送
  • 支持 20+ 语言

上下文管理

  • /context 用量与建议
  • /compact [focus] 智能压缩
  • 95% 自动 compact
  • 1M context（Opus 4.6 Max/Team/Ent）

会话 / SDK

  • claude -c 延续上次对话
  • claude -r "name" 恢复命名会话
  • /btw 轻量新会话
  • claude -p "query" 非交互模式
  • --output-format json 结构化输出
  • cat file | claude -p 管道输入
