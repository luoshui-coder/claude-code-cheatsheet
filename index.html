⌨️ 键盘快捷键

常规控制

  • CtrlC 取消输入/生成
  • CtrlD 退出会话
  • CtrlL 清屏
  • CtrlJ 切换到堆栈输出
  • CtrlR 反向搜索历史
  • CtrlT 在线思路中打开提示
  • CtrlB 后台运行任务
  • CtrlV 粘贴图片
  • EscEsc 回退/撤销

模式切换

  • ShiftTab 循环切换权限模式
  • AltP 切换提问
  • AltT 切换思考模式

执行

  • \Enter 快捷执行
  • CtrlJ 按顺序执行

会话选择器

  • ↑↓ 导航
  • ↵ 展开/折叠

  • B 当前分支

🧰 工作流与技巧

计划模式

  • ShiftTab：Normal → Auto → Plan
  • --permission-mode plan 计划模式启动
  • ultrathink：当前轮次最大努力度
  • AltT：快提/平衡/深入
  • CtrlO：查看思考
  • /effort low · med · high

Git Worktree

  • --worktree name 每个功能独立分支
  • isolation: worktree 让代理跑在独立 worktree
  • sparsePaths 仅检出所需目录
  • /batch 自动创建 worktrees

语音模式

  • /voice 启用语音对话
  • Space（hold）录音，释放发送
  • 20+ 语言：EN/ES/FR/DE/CZ/PL…

上下文管理

  • /context 用量 + 优化建议
  • /compact [focus] 智能压缩
  • Auto-compact 95% 自动触发
  • 1M context / Opus 4.6 (Max/Team/Ent)

会话/SDK

  • claude -c 继续上次对话
  • claude -r "name" 按名称恢复
  • /btw 无上下文开新聊
  • claude -p "query" 非交互式
  • --output-format json 结构化输出
  • cat file | claude -p 管道输入
